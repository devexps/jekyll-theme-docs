# jekyll-theme-docs
A documentation theme for GitHub Pages.

## Usage

### GitHub Pages

Define a `_config.yml` file in your GitHub pages directory and specify this remote theme:

```yaml
remote_theme: devexps/jekyll-theme-docs@main
```
